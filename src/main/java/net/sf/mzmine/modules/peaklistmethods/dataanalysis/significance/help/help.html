Output

Calculated values will be added to CSV-file when a peaklist is exported into CSV with option "Export quantitation results and other information" (see the figure above). Columns STUDENT_P_VALUE, STUDENT_T_VALUE, LOG2_FOLD_CHANGE contain the p-value, t-test, and the logarithmic fold change respectively.
