GNPS Export/Submit

Description

This module exports all files needed for GNPS' feature based molecular networking. This includes the quant table (csv export) and an mgf with all MS2 spectra. Optionally, a new GNPS job is submitted. This job can be cloned into a personal GNPS account.

Instructions

Run this method on any peak list. MS2 spectra need to be associated to the features (usually done in peak deconvolution step).
